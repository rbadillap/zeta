## shadcn/ui Integration

**Features:**
- **CLI:** Initialize and manage the registry and components.
- **Registry:** Schema and structure for component distribution.
- **Components:** Reusable UI primitives for Zeta's interface.

**How Zeta uses shadcn/ui:**
- Uses the CLI for setup and component management.
- Follows the official registry schema.
- Imports and distributes shadcn/ui components.
- See [registry.json](/docs/configuration/registry-json) for more details.
